# warehouse-rover-controller
More complex project, with multi-file Architecture and Error Handling. A robot moves on a 2D grid (X, Y). It must reach a destination without hitting walls and stop if its battery is too low.
### Requirements
    REQ-01: The Rover shall maintain an (X, Y) position.
    REQ-02: The Rover shall move 1 unit North, South, East, or West per command.
    REQ-03: The Rover shall not move outside a 10x10 grid.
    REQ-04: The system shall prevent movement if battery is below 5%.
    REQ-05: Each move shall consume 1% battery.
